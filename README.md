# MyMap

Своя карта мира на Python + Leaflet.js + OpenStreetMap.

## Запуск

```bash
pip install flask
python app.py
```

Открыть: http://localhost:5000

## Возможности

- 🗺 Карта мира с приближением до уровня зданий
- 🔍 Поиск улиц, домов, городов (на русском)
- 📍 Клик по карте — информация об объекте
- 🛰 3 режима: Карта / Спутник / Рельеф
- ◎ Кнопка «Моё местоположение»
- 📋 Копирование координат
